# 🤖 Chainlink BRL Trading Bot

Este é um robô de **trading automatizado real** para o par **CHAINLINK/BRL**, utilizando dados da CoinGecko para decisões de entrada com base em quedas de preço. Também possui um modo de **backtest** para simular e validar a estratégia.

## 🧠 O que ele faz?

- Monitora o par LINK/BRL
- Executa **ordens reais** (em versões com integração via CCXT/Binance)
- Simula estratégias com backtest usando dados históricos
- Envia alertas via Telegram para cada ordem executada

---

## 🔧 Tecnologias Utilizadas

- Node.js
- Axios
- CoinGecko API
- CCXT (para integração com exchanges)
- Telegram Bot API
- Heroku (Deploy)
